# SQL Scripts
SQL was used to explore and analyze the retail sales database before visualization in Power BI. The analysis focused on identifying sales performance, customer behavior, and business trends.  
## SQL Skills Demonstrated  
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- LIMIT
- COUNT()
- SUM()
- AVG()

## Business Questions
The SQL queries answer questions such as:  
- What is the total sales?
- Which products generate the highest sales?
- Which cities have the highest sales?
- Which channel contributes the most sales?
- How many orders are completed?
