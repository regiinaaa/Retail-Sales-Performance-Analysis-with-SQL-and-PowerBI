# Retail-Sales-Performance-Analysis-with-SQL-and-PowerBI
A data analytics project that explores retail sales data using SQL and visualization business insights through an interactive Power BI dashboard.
## Project Overview
This project analyzes retail sales data using SQL for data exploration and Power BI for interactive dashboard visualization.
The objective is to transform raw transactional data into meaningful business insights that support data-driven decision making. The project covers the complete analytics workflow including data exploration, SQL analysis, KPI calculation, dashboard development, and business recommendations.
## Business Problem
Retail businesses generate thousands of transactions every day. Without effective data analysis, it becomes difficult to identify sales trends, top-performing products, customer behavior, and operational performance. This project aims to analyze retail transaction data and present actionable insights through an interactive Power BI dashboard.
## Objectivness
- Analyze retail revenue performance.  
- Identify top-selling products.
- Analyze revenue by category.
- Compare revenue across sales channels.
- Monitor monthly revenue trends.
- Evaluate order status distribution.
- Build an interactive dashboard for business monitoring.
